# Student Registration Form

## Description
A simple Student Registration Form built using HTML5, CSS3, PHP, and MySQL.
It allows users to register student details and stores them in a database.

## Technologies Used
- HTML5
- CSS3
- PHP
- MySQL
- XAMPP

## Features
- Student registration form
- Stores data in MySQL database
- Retrieves data using SELECT query

## How to Run
1. Install XAMPP
2. Start Apache and MySQL
3. Place project inside htdocs folder
4. Create database and table using phpMyAdmin
5. Run `index.html` via localhost
